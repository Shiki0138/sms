## 📝 テストデータの入力方法

### 1. 顧客データの作成
```
例：テスト顧客の作成
- 氏名: テスト太郎
- 電話番号: 090-0000-0001
- メールアドレス: test1@example.com
- 誕生日: 1990年1月1日
```

### 2. 予約データの作成
```
例：テスト予約の作成
- 顧客: テスト太郎
- 日時: 任意の未来日時
- メニュー: カット＆カラー
- スタッフ: 任意のスタッフ
- 備考: テスト予約です
```

### 3. メッセージ送信のテスト
```
例：テストメッセージ
- 宛先: テスト太郎
- 内容: これはテストメッセージです
- 送信方法: LINE（模擬送信）
```

### ⚠️ テストデータ入力時の注意
- 実在の個人情報は絶対に入力しない
- 電話番号は「090-0000-xxxx」形式を推奨
- メールアドレスは「testxx@example.com」形式を推奨

## 🔒 機能制限について

### テスト環境での制限事項

#### 1. 外部送信機能
- ❌ LINE/Instagram実送信（模擬送信のみ）
- ❌ メール送信（画面表示のみ）
- ❌ SMS送信（無効化）

#### 2. データ制限
- ⚠️ 顧客登録: 最大50件
- ⚠️ 予約登録: 最大20件（未来分）
- ⚠️ メッセージ送信: 1日10件まで

#### 3. 高度な機能（有料版で利用可能）
- 🔒 AI分析・予測機能
- 🔒 カスタムレポート作成
- 🔒 データエクスポート機能
- 🔒 外部システム連携設定

#### 4. その他の制限
- ⏱️ セッション時間: 2時間で自動ログアウト
- 📊 レポート生成: 1日3回まで
- 💾 データ保存期間: 30日間

### 1. システム内フィードバック機能
1. 画面右下の「💬 フィードバック」ボタンをクリック

   - 💡 機能改善提案

【再現性】必ず発生する

## ❓ よくある質問

### Q1: テストデータはいつリセットされますか？
**A:** テストデータは30日間保存されます。定期的なバックアップはありません。

### Q2: 複数人で同じアカウントを使用できますか？
**A:** 可能ですが、同時ログインすると片方がログアウトされる可能性があります。

### Q3: スマートフォンでもテストできますか？
**A:** はい、レスポンシブ対応しているため、スマートフォンでもご利用いただけます。

### Q4: テスト期間はいつまでですか？
**A:** テスト期間は3週間を予定しています。詳細な日程は別途ご連絡いたします。

### Q5: 実際の営業で使用できますか？
**A:** テスト環境は実際の営業には使用できません。正式版のリリースをお待ちください。
