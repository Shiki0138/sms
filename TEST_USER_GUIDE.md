## システム概要

SMSは美容室経営に必要な機能を統合した管理システムです。予約管理から顧客管理、売上分析まで、日々の業務を効率化します。

### 🌟 主な機能

| 機能 | 説明 | 利用状況 |
|------|------|----------|
| **顧客管理** | 顧客情報の登録・編集・履歴管理 | ✅ 全機能利用可能 |
| **予約管理** | カレンダー形式での予約管理 | ✅ 全機能利用可能 |
| **メッセージ管理** | 顧客との連絡履歴管理 | ✅ 全機能利用可能 |
| **分析・レポート** | 売上分析・顧客分析 | ✅ 全機能利用可能 |
| **CSVインポート** | 既存データの一括取り込み | ✅ 全機能利用可能 |
| **スタッフ管理** | スタッフ情報・スケジュール管理 | ✅ 全機能利用可能 |

## ログイン・初期設定

### 1. システムアクセス

1. ブラウザでSMSサイトにアクセス
2. ログイン画面でご提供されたアカウント情報を入力
3. 画面上部に「🧪 テスト利用中」バナーが表示されます

### 2. 美容室基本情報の設定

**設定 > 基本設定** から以下を入力：

```
美容室名: [あなたの美容室名]
営業時間: 9:00 - 18:00（調整可能）
定休日: 月曜日（変更可能）
電話番号: [美容室の電話番号]
住所: [美容室の住所]
```

### 3. スタッフ情報の登録

**スタッフ管理** から：

1. 「新しいスタッフを追加」をクリック
2. 名前、役職、スキル等を入力
3. 勤務時間・曜日を設定

### 4. メニューの設定

**設定 > メニュー管理** から：

1. 「新しいメニューを追加」
2. メニュー名、料金、所要時間を入力
3. カテゴリ分類（カット、カラー、パーマ等）

## 機能別ガイド

### 📊 ダッシュボード

**アクセス方法**: 左サイドメニュー「ダッシュボード」

#### 表示内容
- 今日の予約状況
- 売上サマリー
- 新規/既存顧客の比率
- 人気メニューランキング

#### 活用方法
- 朝一番に確認して1日のスケジュールを把握
- 週末に1週間の振り返り
- 月末に月次データを確認

---

### 👥 顧客管理

**アクセス方法**: 左サイドメニュー「顧客管理」

#### 顧客登録
1. 「新規顧客登録」ボタンをクリック
2. 基本情報を入力：
   ```
   氏名: [顧客名]
   電話番号: [連絡先]
   メールアドレス: [任意]
   生年月日: [任意・誕生日特典で活用]
   住所: [DMやアンケートで活用]
   ```
3. 特記事項・アレルギー情報等を「備考」に記入
4. 「保存」で完了

#### 顧客情報の管理
- **検索機能**: 名前・電話番号で素早く検索
- **履歴確認**: 過去の来店履歴・施術メニュー
- **メモ機能**: 次回来店時の参考情報を記録

#### CSVインポート機能
既存の顧客データがある場合：

1. 「CSVインポート」をクリック
2. テンプレートをダウンロードして形式確認
3. 既存データをCSV形式で準備
4. ファイルをアップロードして一括登録

**対応形式**: ホットペッパービューティー形式

---

### 📅 予約管理

**アクセス方法**: 左サイドメニュー「予約管理」

#### 表示切替
- **週表示**: 1週間のスケジュールを一覧
- **日表示**: 1日の詳細スケジュール
- **月表示**: 月間カレンダー

#### 新規予約の作成
1. カレンダー上の希望時間をクリック
2. 予約情報を入力：
   ```
   顧客: [既存顧客を選択または新規作成]
   メニュー: [提供メニューから選択]
   担当スタッフ: [担当者を指定]
   所要時間: [メニューに応じて自動設定]
   料金: [メニューに応じて自動設定]
   ```
3. 特記事項があれば「備考」に記入
4. 「予約を作成」で完了

#### 予約の管理
- **ステータス変更**: 予約確定 → 来店 → 完了
- **キャンセル処理**: キャンセル理由を記録
- **リスケジュール**: ドラッグ&ドロップで時間変更

---

### 💬 メッセージ管理

**アクセス方法**: 左サイドメニュー「メッセージ」

#### デモモードでの機能
- 📖 **閲覧のみ**: 過去のメッセージ履歴確認
- 🚫 **送信制限**: LINE・Instagram送信は無効

#### 活用方法
- 顧客との過去のやり取りを確認
- 要望やクレーム履歴の把握
- フォローアップの必要性判断

---

### 📈 分析・レポート

**アクセス方法**: 左サイドメニュー「分析・レポート」

#### 利用可能な分析（デモモード）
1. **売上分析**
   - 日別・月別売上推移
   - メニュー別売上構成
   - スタッフ別売上実績

2. **顧客分析**
   - 新規・既存顧客比率
   - リピート率
   - 顧客年代別分布

3. **予約分析**
   - 時間帯別予約状況
   - 曜日別傾向
   - キャンセル率

#### データエクスポート
- 📊 **CSVダウンロード**: Excel等で詳細分析
- 📄 **PDFレポート**: 会議資料として活用

---

### ⚙️ 設定管理

**アクセス方法**: 左サイドメニュー「設定」

#### 営業設定
```
営業時間: 開店・閉店時間の設定
定休日: 週の定休日設定
休暇設定: 臨時休業日の登録
予約間隔: 予約受付の最小単位（15分/30分等）
```

#### メニュー管理
- **カテゴリ分類**: カット、カラー、パーマ、トリートメント等
- **料金設定**: 基本料金・オプション料金
- **所要時間**: 予約スケジュールの自動計算用

#### スタッフ管理
- **基本情報**: 名前、役職、連絡先
- **勤務スケジュール**: 出勤日・時間
- **スキル設定**: 対応可能メニュー

## よくある質問（FAQ）

### Q1: このシステムは本番環境ですか？
**A**: はい、本番環境で動作しています。ただし現在はテスト期間中のため、フィードバックをお願いしています。

### Q2: 複数のスタッフで同時に利用できますか？
**A**: はい、複数端末からの同時アクセスが可能です。リアルタイムで予約状況が同期されます。

### Q3: 既存のPOSレジとの連携は可能ですか？
**A**: CSV形式でのデータ連携が可能です。詳細についてはお問い合わせください。

### Q4: スマートフォンアプリはありますか？
**A**: 現在はWebアプリのみですが、スマートフォン・タブレットに最適化されています。

### Q5: データのバックアップは取れますか？
**A**: CSVエクスポート機能でデータのダウンロードが可能です。

### Q6: 実際にLINEやInstagramに送信されますか？
**A**: はい、設定が正しく行われていれば実際に送信されます。テスト期間中は十分ご注意ください。
