## 🔧 技術スタック

### バックエンド
- **言語**: TypeScript
- **フレームワーク**: Express.js
- **データベース**: SQLite (開発) / PostgreSQL (本番)
- **ORM**: Prisma
- **認証**: JWT + 2FA
- **リアルタイム**: Socket.io
- **キャッシュ**: Redis
- **ジョブキュー**: Bull Queue
- **プラン制限**: ミドルウェアベース機能制御
- **使用量監視**: リアルタイム制限チェック

### フロントエンド
- **フレームワーク**: React 18
- **言語**: TypeScript
- **UI**: Tailwind CSS
- **状態管理**: Context API + Hooks
- **ビルドツール**: Vite

### インフラ・DevOps
- **コンテナ**: Docker + Docker Compose
- **プロキシ**: Nginx
- **監視**: Prometheus + Grafana
- **ログ**: Winston
- **CI/CD**: GitHub Actions
